### 2. 스테이션 심화 분석을 왜 하나요?
- 목표는 유저 개인화 서비스 개발
- 하지만, 유저별 사용 이력 데이터는 존재하지 않음.
- 스테이션 개별 분석은 유저 사용 데이터에 인사이트를 줄 것이라 기대
- 공공자전거 이용 패턴을 더 잘 이해하고 사용자별 분석에 중요한 통찰력을 얻기 위함.
